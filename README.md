# Mantle Calendar

> [!CAUTION]
> **Status: Beta (v1.0.0)**  
> This plugin is currently in Alpha. Features and UI are subject to change.

Mantle Calendar provides a streamlined calendar interface directly within Obsidian. It's designed to help you manage time-sensitive tasks and events without leaving your vault.

## Features
- **Event Management**: Easily track tasks and events on a daily or monthly view.
- **Note Integration**: Links calendar events to your Markdown notes.
- **Zenith Optimized**: Perfectly aligned with the [Zenith theme](https://github.com/carnalMATRIX/obsidian-mantle-zenith) for a consistent workspace aesthetic.

## Installation

### Manual Installation
1. Download the `main.js`, `manifest.json`, and `styles.css` from the latest release.
2. Create a folder named `mantle-calendar` in your vault's `.obsidian/plugins/` directory.
3. Move the downloaded files into that folder.
4. Restart Obsidian and enable **Mantle Calendar** in **Settings > Community plugins**.

## Development

To modify this plugin:
1. Navigate to this directory in your terminal.
2. Install dependencies: `npm install`
3. Build the plugin: `npm run build`
4. For active development, use: `npm run dev`

This plugin is built with TypeScript and esbuild.
